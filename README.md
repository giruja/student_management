# Student Management System

# Description
This is a Python-based Student Management System that allows users to manage student records.

# Features
- Add student
- View all students
- Search student
- Update student details
- Show Rank
- Save data to file
- Load data from file

# Technologies Used
- Python
- JSON

# How to Run
1. Install Python
2. Run the file:
   python filename.py
